# Product

Menu Translator is a browser-based mobile-first web application.

Primary flow:
1. User uploads or takes a photo of a restaurant menu.
2. The app sends the image to Gemini.
3. Gemini extracts and translates the menu.
4. The user browses the translated menu.
5. The user adds dishes to a cart.
6. The user shows a waiter-friendly order summary.

Target users:
- tourists
- international students
- travellers
- people ordering from menus they cannot read

Primary device:
- mobile browser

Desktop behaviour:
- show a centred mobile-style app shell
