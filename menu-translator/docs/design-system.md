# Design System

Design direction:
- premium mobile utility
- modern food and travel assistant
- clean, warm, and trustworthy
- inspired by Airbnb, Uber Eats, Apple Wallet, Linear

Use:
- shadcn/ui components
- Tailwind CSS
- Lucide icons
- Framer Motion

Patterns:
- cards
- sticky bottom actions
- drawers
- sheets
- skeleton loaders
- clear empty states
- clear error states

Motion:
- subtle
- fast
- functional
- no excessive animation

Accessibility:
- minimum 44px touch targets
- readable text
- strong contrast
- keyboard accessible controls
- respect reduced motion preferences
